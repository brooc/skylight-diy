# Feature: Kiosk Mode

## Goal

Make a cheap tablet feel like a dedicated family appliance.

## MVP

- Large landscape layout
- Fullscreen browser support
- No hover-dependent interactions
- Big touch targets
- Low visual clutter
- Visible sync status
- Safe refresh behavior

## Fire Tablet considerations

- Silk Browser compatibility
- Fullscreen limitations
- Stay-awake behavior
- Charging cable placement
- Screen burn-in mitigation
- Night dimming
- Wall mount vs countertop stand

## Later

- Motion wake
- Ambient mode
- Remote reload
- Health check
- Admin PIN
