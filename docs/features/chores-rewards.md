# Feature: Chores and Rewards

## MVP

- People in household
- Chore list by person
- Recurring chores
- Complete chore from tablet
- Award points or stars
- Show reward balance
- Parent/admin can define rewards

## Interaction model

Kids should be able to:

- See their chores
- Tap to complete
- See stars/points earned
- See available rewards

Adults should be able to:

- Create chores
- Assign chores
- Adjust points
- Approve or undo completions if needed
- Redeem rewards

## Open questions

- Are chore completions trusted by default?
- Should some chores require approval?
- Do points expire?
- Can chores be shared by multiple people?
- Should chores have due windows?
