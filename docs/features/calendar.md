# Feature: Calendar

## MVP

- Day view
- Week view
- Multiple calendar sources
- Person/color mapping
- Event title, time, location
- Manual refresh

## Later

- Add events from tablet
- Edit events
- RSVP awareness
- Conflict warnings
- Natural language event creation
- Calendar provider plugins

## Design questions

- Should we sync into our database or read live from providers?
- Should household member calendars be first-class objects?
- How do we handle recurring events?
- What is the offline behavior?
