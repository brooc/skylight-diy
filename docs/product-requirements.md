# Product Requirements

## Target users

- Families with children
- Shared households
- People who want a visible home schedule
- Makers who want a DIY alternative to dedicated calendar appliances
- Privacy-conscious users who prefer self-hosted tools

## Target hardware

Initial target:

- Amazon Fire HD 10 tablet
- Browser-based app
- Always-powered kiosk display

Future targets:

- Android tablets
- iPads
- Raspberry Pi touchscreen displays
- Wall-mounted monitors
- Desktop browser

## MVP requirements

### Calendar

- Show today and upcoming events
- Support Google Calendar first
- Support multiple family member calendars
- Color-code people/calendars
- Provide day and week views
- Show event title, time, location, and assigned person

### Chores

- Create chores
- Assign chores to people
- Support recurring chores
- Mark chores complete from the tablet
- Show overdue chores
- Show daily chore list by person

### Rewards

- Award points or stars for completed chores
- Track balances per person
- Allow rewards to be defined by parent/admin
- Allow rewards to be redeemed manually

### Meal planning

- Show dinner plan for the week
- Add simple meals to dates
- Keep reusable meal ideas
- Generate a grocery list manually or semi-automatically

### Kiosk mode

- Run fullscreen on a Fire Tablet
- Avoid accidental navigation away from the app
- Keep the screen awake while charging
- Provide a low-brightness or sleep display mode

### Magic Import, later MVP+

- Accept text pasted into the app
- Extract candidate calendar events
- Show extracted events for review
- Create events only after user approval

## Non-goals for the first version

- Replacing every calendar app
- Fully automatic event creation without review
- Native mobile apps
- Complex household permissions
- Payment systems or hosted subscriptions
- Full clone of any proprietary product
