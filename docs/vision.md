# Project Vision

## What we are building

Daymark is an open-source family command center that runs on hardware people already own.

It should help a household answer:

- What is happening today?
- Who needs to do what?
- What is for dinner?
- What do we need to buy?
- Did the kids do their chores?
- Can I turn this flyer, email, PDF, screenshot, or schedule into calendar events without typing everything manually?

## Why this exists

Dedicated family calendar appliances are useful, but expensive and closed. Many families already have old tablets, spare monitors, or inexpensive Android devices that can handle the display and touch interface.

This project should make the useful part accessible, hackable, and extensible.

## Principles

1. **Runs on cheap hardware**  
   The first target is a Fire HD 10 tablet.

2. **Family-first UX**  
   The interface should be understandable by kids, partners, grandparents, and tired parents.

3. **Open and hackable**  
   It should be easy to self-host, customize, fork, and contribute to.

4. **Privacy-conscious by default**  
   Family schedules, children’s names, school PDFs, and household routines are sensitive.

5. **Useful before magical**  
   Calendar, chores, and planning must work before we add advanced AI import features.

6. **No forced subscription**  
   Optional hosted services can exist later, but the core should be usable without recurring fees.
